# Afrodita Lencería — v5 Premium Visual

Versión enfocada en optimización visual premium, manteniendo la base ecommerce sin Mercado Pago.

## Incluye

- Home rediseñada con estética premium.
- Navbar premium responsive.
- Footer completo con accesos rápidos.
- Cards de producto más elegantes.
- Catálogo mejor presentado.
- Detalle de producto más visual.
- Panel de selección de talle/color mejorado.
- Carrito con mejor UX.
- Checkout manual más profesional.
- Admin dashboard visualmente mejorado.
- Páginas legales, privacidad, cambios, envíos y contacto.
- Supabase + Firebase ya preparados.
- Sin Mercado Pago por ahora.

## Ejecutar

```bash
npm install
npm run dev
```

## Supabase

Ejecutar:

```txt
supabase/schema.sql
```

Crear bucket público:

```txt
products
```

## Variables

Copiar `.env.example` a `.env.local` y completar Supabase/Firebase.
